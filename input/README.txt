파일 목록
1. 예측 분석 및 모델링 기초 데이터
	1-1. train_y.csv: 예측 타겟인 발전단지 총 발전량 데이터 (2020-01-01 01:00 ~ 2022-12-31 24:00)
	1-2. windfarm_metadata.xlsx: 영광 및 경주 풍력발전단지 정보
	1-3. train/test_ldaps_yeonggwang.pkl: 영광풍력단지 발전량 예측을 위해 기본 제공되는 feature인 LDAPS 기상예보
	1-4. train/test_ldaps_gyeongju.pkl: 경주풍력단지 발전량 예측을 위해 기본 제공되는 feature인 LDAPS 기상예보
		- 해당 기상 예보는 모두 참고사항 2를 고려해, 타겟일 하루 전 15시 경에 예보된 타겟일의 24시간 데이터
		- 추가 feature를 사용할 경우, 타겟일 하루 전 15시 이전의 기상예보 데이터 혹은 불변하는 기타 환경/지형 변수만 활용 (참고사항 2)
2. 제출용 데이터 포맷, submission_format.csv
3. 분석용 데이터인 풍력발전단지 SCADA
	3-1. 영광풍력 SCADA
	3-2. 경주풍력 SCADA
4. 기본 예제 및 활용 권장 라이브러리
	4-1. 예제_배포용.ipynb: 기본 제공 데이터 및 라이브러리를 활용한 발전량 예측 및 검증 절차 (필독 권장)
	4-2. windpowerlib-0.2.1-py3-none-any.whl: 활용 권장하는 라이브러리 whl 파일
	4-3. Power_curves_20210323.xls: 풍력발전터빈 파워커브 데이터베이스 (출력-풍속 관계 커브)


참고사항
1. 학습/분석 및 예측 제출 기간
	1-1. 학습/분석용 데이터 기간: 2020-01-01 ~ 2022-12-31
	1-2. 시험용 데이터 기간(제출): 2023-01-01 ~ 2023-12-31
2. 재생에너지는 하루 뒤 24시간을 당일 15시 경에 예측하는 것이 제도로 운영되고 있음 (자세한 규칙은 KPX 재생에너지 발전량 예측제도 참고)
	- 본 경연대회에서 동일한 규칙을 적용해 예측 타겟 당일의 기상관측 혹은 기상예보 재해석장을 활용하는 경우 실격처리
	- 예: 2023-09-16의 24시간 예측을 위해 2023-09-15 15시 경에 발령된 기상예보만 사용
3. 데이터 추출 과정 중 오류로 인한 일부 시점 누락이 있을 수 있으나, 시험 기간 내 누락 시점은 채점 시 반영되지 않음
4. 풍력발전기의 관한 기본적인 설명은 에너지공단 페이지(https://www.knrec.or.kr/biz/korea/intro/kor_wind.do) 참고
5. 풍력발전터빈은 모두 UNISON 사의 것으로, SCADA의 컬럼에 관한 자세한 설명은 UNISON 사의 스펙시트 등 참고
