## 평가 지표 :
- NMAE (Normalized Mean Absolute Error):  
    - 값이 작을수록 잘 예측한 것
```python
import numpy as np

def NMAE(true, pred):
    mae = np.mean(np.abs(true - pred))
    score = mae / np.mean(np.abs(true))
    return score
```
